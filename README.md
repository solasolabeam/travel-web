## 프로젝트 소개
> 다채롭고 아름다운 국내여행지 찾기

저는 국내 다양한 지역을 여행하는 것을 좋아합니다. 대자연의 아름다움부터 도시의 매력적인 풍경까지, 각 지역마다 고유한 특색과 문화를 경험하는 것이 저에게 큰 즐거움이기에 이러한 장소를 지역이나 카테고리별로 탐색하는 사이트를 만들게 되었습니다
